# NSFC LaTeX

非官方国家自然科学基金 LaTeX 模板，旨在为习惯使用 LaTeX 的研究人员提供便利。

习惯使用 LaTeX 的用户可以轻松地复用已发表论文中的图表参考文献等内容。

本模板经过精心排版，力求与原版 Word 模板高度一致。

> [!WARNING]  
> 请注意，尽管已有多个成功申请案例使用了本模板，但我们无法对使用本模板的结果提供任何保证。
